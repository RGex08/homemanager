## ğŸ¯ Common Tasks

### Add a New Property
1. Go to **Properties** page
2. Fill "Add New Property" form
3. Click "Add property"
4. Property appears in sidebar

### Add a Unit
1. Stay on Properties page
2. Fill "Add Unit to Property" form
3. Select property from dropdown
4. Enter unit label and rent
5. Click "Add unit"

### Create a Lease
1. Click unit in sidebar
2. Go to **Tenants** tab
3. Add tenant (if not already added)
4. Go to **Leases** tab
5. Fill lease form:
   - Select tenant
   - Enter dates
   - Enter rent amount
6. Click "Create lease"
7. Payments automatically generated!

### Track Rent Payment
1. Navigate to unit with active lease
2. Go to **Leases** tab
3. Scroll to "Payment Tracking" section
4. Click "Mark Paid" for received payments
5. Dashboard updates automatically

### Log Maintenance Issue
1. Navigate to affected unit
2. Go to **Maintenance** tab
3. Fill maintenance form:
   - Describe issue
   - Select category
   - Link to feature (optional)
4. Click "Create request"

## ğŸ†˜ Quick Troubleshooting

### "Nothing happens when I click"
- Check browser console (F12)
- Verify JavaScript is enabled
- Try hard refresh (Ctrl+F5)

### "Data disappeared"
- Did you clear browser data?
- Using Private/Incognito mode?
- Different browser?

### "Can't delete property"
- Properties with units cannot be deleted
- Delete all units first
- Then delete property

### "Can't delete unit"
- Units with active leases cannot be deleted
- End the lease first
- Then delete unit

### "Forgot password"
- Click "Forgot password?"
- Enter email
- Follow reset flow
- (Prototype: just sets new password)
